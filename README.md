# valen-bici

**valen-bici** es un proyecto rápido para practicar la conexión a APIs y JavaScript. Fue desarrollado con ayuda de IA como una forma creativa y divertida de aprender.

## Tecnologías
- HTML
- CSS
- JavaScript

## Cómo usarlo
Solo abre el proyecto en tu navegador desde el siguiente link:

[Ver valen-bici en GitHub Pages](https://bogstar21.github.io/valen-bici/)

## Descripción
Este proyecto permite experimentar con la conexión a APIs y la manipulación de datos en tiempo real usando JavaScript. Es ideal para quienes quieren aprender de manera práctica y entretenida.
